01 / clipdrop
Serverless · Live
ClipDrop

Your browser is the pendrive. Create a room, drop your files, share the code — access them instantly on any device. No account, no install, no cloud storage. Files auto-delete when the room expires.

  • Room-code based sharing — no login required
  • Direct S3 upload via presigned URLs — files never touch the server
  • User-controlled expiry — 10 min, 30 min, 1hr, 6hr, 24hr, or on tab close
  • Auto file deletion via S3 lifecycle rules + DynamoDB TTL
  • Supports any file format up to 500 MB per file
  • Live countdown timer, per-file progress bars, download all
